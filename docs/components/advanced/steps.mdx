# 步骤条
引导用户按照流程完成任务的导航条。

## 何时使用
* 当任务复杂或者存在先后关系时，将其分解成一系列步骤，从而简化任务。
## 组件示例
import Iframe from 'components/Iframe'

<Iframe url="https://www.aliwork.com/developer/steps-v2?isRenderNav=false" />
