LinkBlock 链接块

链接块容器，功能同链接组件，不过可以嵌入其他内容，例如图片等。

何时使用​

  • 切换访问当前应用的其他页面；
  • 跳转到外部页面。

是否禁用，禁用后点击链接块不会进行页面跳转

boolean

false

是否新开页面

boolean

false

page

内部页面Uuid，当 type 属性为page时生效
