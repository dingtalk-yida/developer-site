图片裁剪上传（高阶）

info
  • 该组件在自定义组件案例 - 图片裁剪上传（中阶）的基础上做了进步的封装，从而较少用户使用成本。
  • 注意：该组件仅供学习使用，并不是官方提供的标准组件。

案例背景​

  • 宜搭组件发布了一个自定义组件的功能，支持用低代码的方式做一个组件，但并不支持表单组件，只能做些展示型的组件。
  • 但文档上提示可以用已有的表单做存储，利用这种方式实现了个图片剪裁上传的组件，通过生成一个新的表单来存储数据，从而模拟出一个伪表单组件。
