## 操作步骤

通过自定义页面与按钮组件点击事件的配合，调用浏览器自带的打印功能，实现页面的打印。

### 步骤 1：创建并配置自定义页面

准备容器组件、按钮组件和想要打印内容的组件。将需要打印的内容放置在容器组件内，设置容器组件属性。（如下图所示）
![创建并配置自定义页面](https://img.alicdn.com/imgextra/i1/O1CN01BzkWL91te9pVUw9PF_!!6000000005926-2-tps-1439-571.png_.webp)

### 步骤 2：实现打印功能

为了页面美观，表格组件的操作列、顶部操作和分页都设置了不显示。

> 表格内容的详细配置可以参考：[自定义页面表格实现数据管理页功能](https://www.yuque.com/yuxinxin-tvkfq/ceqhg5/uxe823)

按钮组件设置 onclik 点击动作，实现点击按钮时调用浏览器的功能，打印指定区域。（如下图所示）
![打印指定区域](https://img.alicdn.com/imgextra/i3/O1CN01MTSmto1lqxyjzNs5h_!!6000000004871-2-tps-1438-807.png_.webp)

> 下述代码可直接复制到 JS 面板，注意：需要替换容器组件唯一标识。

```js
export function onClick(){
  //绘制需要打印的区域为容器组件内的内容
  const printPart = document.getElementById('div_ktdxeg0h');
  window.document.body.innerHTML = "";
  window.document.body.appendChild(printPart);
  let oldstr = window.document.body.innerHTML;
  //调用print()方法去打印定义的网页区域
  window.print();
  document.body.innerHTML = oldstr;
  location.reload();
}
```

调用完成后自定义打印预览中有页眉和页脚，可以取消勾选将其去掉，去掉后可以选择另存为 pdf 去保存。（如下如所示）
![设置打印预览](https://img.alicdn.com/imgextra/i2/O1CN01GSIlbL1F6Bp6Lk7xU_!!6000000000437-2-tps-1436-802.png_.webp)
